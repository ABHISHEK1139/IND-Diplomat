# Frontend Folder Guide

Active UI files:

- index.html: minimal chat layout
- styles.css: responsive minimalist design
- app.js: chat workflow, history, API calls

Legacy/support files kept for compatibility:

- analyst.js: analyst workstation extension script
- server.py: standalone UI proxy server
- styles_fixes.css: legacy style overrides

Primary app entrypoint uses app_server.py at repository root.
